Roles:

1. System admin: 
- Creates organization login details for enterprises. 
- Creates login details for roles.

2. Salon Organization Admin:
- Edit his profile
- Add salon services
- View requests raised by customer. Accept/reject those requests. If accepted, assign them to a beautician.

3. Home Services Organization Admin:
- Edit his profile
- Add home services services
- View requests raised by customer. Accept/reject those requests. If accepted, assign them to a home service technician.

4. Marketing Organization Admin:
- Edit his profile
- Create a feedback form
- View the feedback given.

5. House Painting Organization Admin: 
- Edit his profile
- Add painting services
- View orders raised by customer. Accept/reject those orders. If accepted, assign them to a painter.

6. Pest Control Organization Admin:
- Edit his profile
- Add pest control services
- View requests raised by customer. Accept/reject those requests. If accepted, assign them to a pest control technician.

7. Customer:
- Raise requests for Salon and Home Services organisations. He can add the services to cart and then place requests.
- Order a painting package from a painter's organisation.
- Call a pest control technician for a selected pest control service from a selected pest control organisation.

8. Beautician:
- View the salon and home service requests assigned to him. Process them and then complete the request.

8. Home Service Technician:
- View the salon and home service requests assigned to him. Process them and then complete the request.

9. Painter:
- View the orders assigned to him. Process and then complete the order.

10. Pest Control Technician:
- View the pest control service requests assigned to him. Process and complete the request
